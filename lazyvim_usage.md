# LazyVim 跳转与书签会话记录

记录时间：2026-03-23

## 1. 函数跳转后如何返回
在 LazyVim 中，函数定义跳转后想回到原位置，通常使用 Vim/Neovim 的 jumplist：

- 返回上一个跳转位置：`Ctrl-o`
- 再向前跳回来：`Ctrl-i`
- 查看跳转记录：`:jumps`

适用场景包括：
- `gd`
- `gD`
- `gr`
- 通过 LSP 执行 definition/reference 等跳转

说明：
- `Ctrl-o` 用于 jump back
- `Ctrl-i` 用于 jump forward

## 2. LazyVim 是否支持“书签”
可以，最基础也最常用的是使用 Vim 内置的 `mark`。

### 添加与跳转
- 添加书签：`m{字母}`
  - 示例：`ma` 表示在当前位置打一个名为 `a` 的标记
- 跳转到书签：`'{字母}` 或 `` `{字母}` ``
  - `'a`：跳到 `a` 所在行
  - `` `a` ``：精确跳到 `a` 所在列
- 查看所有标记：`:marks`

### 补充说明
- 小写字母 `a-z`：常用于普通标记
- 大写字母 `A-Z`：更适合跨文件跳转，可视作全局书签
- 同一个字母重复打标会覆盖旧位置

### 相关常用位置跳转
- 返回上次跳转前位置：`` `` `` 或 `''`
- 上次编辑位置：`'.`
- 文件上次退出位置：`'"`

## 3. 如果想要更像 IDE 的书签体验
除了内置 `mark`，还可以考虑插件：

- `harpoon`：偏向快速标记文件并跳转
- `arrow`
- `marks.nvim`
- `bookmark.nvim`

如果只是做轻量定位，内置 `mark` 已经足够高效。

## 4. 建议
对于日常使用，优先记住以下快捷键即可：

- `Ctrl-o`：跳回
- `Ctrl-i`：跳前
- `ma`：添加书签
- `'a`：跳到书签所在行
- `` `a` ``：跳到书签精确位置
- `:jumps`：查看跳转历史
- `:marks`：查看所有书签/标记
